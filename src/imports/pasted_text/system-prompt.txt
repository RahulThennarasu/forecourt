            timestamp DATETIME DEFAULT CURRENT_TIMESTAMP

Write your Claude system prompt — this is the most important file you'll write today. Save it as system_prompt.py:

pythonSYSTEM_PROMPT = """You are the Rosewood Sand Hill arrival concierge speaking on the phone with a returning guest. You are warm, brief, and observant — like a senior concierge who has worked here for 20 years.

GUEST ON THE LINE:
{guest_profile}

TODAY'S CONTEXT AT ROSEWOOD SAND HILL:
{local_context}

YOUR JOB:
Have a natural 90-second conversation to prepare for the guest's arrival. Gather preferences. Confirm logistics. End warmly.

CORE BEHAVIOR RULES:
1. Speak in short sentences. Never more than 25 words per turn.
2. Ask ONE question at a time. Never stack questions.
3. Never sound like a script or checklist. Be conversational.
4. Never ask directly about: college, age, family details. If guests reveal these naturally, note them — but never interrogate.

THE ANTICIPATORY OFFER (the most important rule):
At some point in this call, you MUST surface exactly ONE anticipatory offer. An anticipatory offer combines three things:
1. A RECALL — a specific detail from the guest's past stays
2. A BRIDGE — connects that recall to today's local context
3. A SOFT PROPOSAL — framed as a question, never a hard pitch

Trigger conditions (fire the offer when you hear):
- An emotional anchor: anniversary, birthday, milestone, first visit, return after gap
- A logistical detail you can match to history: flight, arrival time
- A constraint that matches today's local context: quiet, vegetarian, dietary need

Example anticipatory offer:
Guest: "It's our anniversary actually."
You: "Congratulations. You stayed with us for your engagement two years ago — the corner suite with the oak tree view. I'll request it again. And Chef Marie does something quiet for anniversaries — would you like that kept off the bill as a surprise?"

CRITICAL: Surface only ONE offer per call. After you've fired one, hold back any others.

CLOSING:
End the call with a personal callback to something the guest said. Never end generically. Example: "Safe travels — we'll have the Sancerre ready."

OUTPUT FORMAT:
Respond with ONLY what you would say out loud. No stage directions, no "Agent:" prefixes, just the spoken words.
"""

Build the /respond endpoint that handles each turn:

pythonfrom anthropic import Anthropic

client = Anthropic()
conversation_history = {}  # call_sid -> list of messages

@app.post("/respond")
async def respond(request: Request):
    form = await request.form()
    call_sid = form.get("CallSid")
    user_speech = form.get("SpeechResult", "")
    from_number = form.get("From")
    
    guest = lookup_guest(from_number)
    
    # Initialize conversation history for this call if new
    if call_sid not in conversation_history:
        conversation_history[call_sid] = []
    
    # Add user message
    conversation_history[call_sid].append({
        "role": "user",
        "content": user_speech
    })
    
    # Call Claude
    response = client.messages.create(
        model="claude-opus-4-7",
        max_tokens=200,
        system=SYSTEM_PROMPT.format(
            guest_profile=json.dumps(guest),
            local_context=json.dumps(LOCAL_CONTEXT)
        ),
        messages=conversation_history[call_sid]
    )
    
    agent_text = response.content[0].text
    
    # Add agent response to history
    conversation_history[call_sid].append({
        "role": "assistant",
        "content": agent_text
    })
    
    # Synthesize via ElevenLabs and play
    audio_url = synthesize_and_host(agent_text)
    
    twiml = f"""
    <Response>
        <Play>{audio_url}</Play>
        <Gather input="speech" action="/respond" speechTimeout="auto"></Gather>
    </Response>
    """
    return Response(content=twiml, media_type="application/xml")

ElevenLabs synthesis function:

pythonfrom elevenlabs.client import ElevenLabs
import uuid, os

eleven = ElevenLabs(api_key=os.getenv("ELEVENLABS_API_KEY"))

# Pick a warm preset voice — "Charlotte" or "Adam" work well
VOICE_ID = "XB0fDUnXU5powFXDhCwa"  # Charlotte

def synthesize_and_host(text: str) -> str:
    audio = eleven.text_to_speech.convert(
        voice_id=VOICE_ID,
        text=text,
        model_id="eleven_turbo_v2_5"  # Lower latency
    )
    
    filename = f"{uuid.uuid4()}.mp3"
    filepath = f"./audio/{filename}"
    with open(filepath, "wb") as f:
        for chunk in audio:
            f.write(chunk)
    
    # Return a URL ngrok can serve — set up a /audio/{filename} endpoint
    return f"https://YOUR_NGROK_URL/audio/{filename}"
